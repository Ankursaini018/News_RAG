About this project

Built for exploring news with AI.

This project combines retrieval-based question answering with a lightweight web interface so you can search through BBC news content in a natural way.

How it works

The assistant loads news content, chunks it into smaller pieces, embeds those chunks, and retrieves the most relevant passages when you ask a question. The answer is then assembled using the retrieved context.

Retrieval

Relevant content is pulled from a vector index built from processed news articles.

Generation

The model uses the retrieved context to answer your question in a helpful, grounded way.
