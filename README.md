# 📰 News RAG Chatbot

A Retrieval-Augmented Generation (RAG) application that extracts news from websites and answers user questions using **LangChain**, **FAISS**, **HuggingFace Embeddings**, and **Groq LLM**.

---

## 🚀 Features

- 🌐 Load news from a website
- ✂️ Split articles into chunks
- 🧠 Generate embeddings
- 📦 Store vectors using FAISS
- 🔎 Semantic similarity search
- 💬 Question Answering with Groq

---

## 🛠 Tech Stack

- Python
- LangChain
- FAISS
- HuggingFace Embeddings
- Groq API
- Jupyter Notebook
